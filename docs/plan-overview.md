# Sleep Care Project Overview

## 프로젝트 한 줄 정의
수험생의 공부 중 졸음과 실제 수면 패턴을 함께 분석해 더 좋은 수면 루틴을 제안하는 통합 케어 프로젝트.

## 제품 구성
- 라즈베리파이5 디바이스: 카메라 기반 졸음 감지, LCD 표시, 스피커 알림, 블루투스 전송
- 스마트폰 앱: 데이터 허브, 분석, 시각화, 수면 스케줄 제안
- 스마트워치 앱: 수면 데이터 수집 및 스마트폰 동기화

## 핵심 사용자 가치
- 공부 중 졸음 상태를 실시간으로 파악
- 실제 수면 패턴과 졸음 패턴을 함께 분석
- 학습 플랜과 시험 날짜에 맞춘 수면 스케줄 제안

## 기본 사용자 흐름
1. 사용자가 스마트폰 앱에서 프로필을 설정한다.
2. 라즈베리파이와 스마트워치를 스마트폰 앱에 연결한다.
3. 라즈베리파이는 공부 중 졸음을 감지한다.
4. 스마트워치는 수면 데이터를 기록한다.
5. 스마트폰 앱은 두 데이터를 수집하고 분석한다.
6. 앱은 사용자에게 수면 스케줄과 생활 가이드를 제안한다.

## 시스템 역할 분리
- 라즈베리파이: 실시간 감지와 즉시 반응
- 스마트워치: 수면 데이터 수집
- 스마트폰 앱: 저장, 분석, 추천, 사용자 경험 제공

## 모바일 앱 기술 스택 요약
- 앱 본체: Kotlin
- UI: Jetpack Compose + Material 3
- 아키텍처: MVVM + Repository + UDF(단방향 데이터 흐름)
- 상태관리 / 비동기 / DI: ViewModel + Kotlin Coroutines + Flow + Hilt
- 로컬 저장: Room + DataStore
- 백그라운드 작업: WorkManager
- 라즈베리파이 통신: BLE(GATT)
- 초기 페어링: CompanionDeviceManager 우선 검토
- 워치 수면 데이터 연동: 1차 선택은 Health Connect

## 핵심 의사결정 포인트
- 모바일 앱 기술 스택
- 워치 대상 플랫폼
- 라즈베리파이 졸음 감지 방식
- Bluetooth 통신 설계
- 데이터 저장 구조

## MVP 범위
- 라즈베리파이 기본 졸음 감지
- 스마트워치 수면 데이터 연동
- 모바일 앱 데이터 수집 및 조회
- 간단한 수면 스케줄 제안
