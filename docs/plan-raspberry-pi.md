# Raspberry Pi Plan

## 목표
- 사용자의 졸음 상태를 실시간으로 감지한다.
- 졸음이 감지되면 즉시 알림을 제공한다.
- 감지 결과를 스마트폰 앱으로 안정적으로 전송한다.

## 주요 기능
- 카메라 영상 수집
- 얼굴 또는 눈 상태 감지
- 졸음 판단 로직
- LCD 상태 표시
- 스피커 경고음 재생
- 블루투스 페어링 및 데이터 전송
- 기기 상태 확인용 간단 UI

## 모바일 연동 기준
- 스마트폰 앱과의 통신은 BLE(GATT) 기준으로 맞춘다.
- 전송 데이터는 졸음 이벤트 중심의 경량 payload로 설계한다.
- 초기 연결 UX는 모바일 앱에서 CompanionDeviceManager 검토를 전제로 한다.

## 세부 모듈
- 영상 입력 모듈
- AI 추론 모듈
- 졸음 이벤트 판단 모듈
- 알림 출력 모듈
- Bluetooth 통신 모듈
- 기기 설정 및 로그 모듈

## 구현 단계
1. 카메라 입력 안정화
2. 얼굴 및 눈 상태 감지 구현
3. 규칙 기반 졸음 감지 MVP 구현
4. LCD 출력 및 경고음 연결
5. Bluetooth 데이터 전송 구현
6. 실제 환경 테스트 후 모델 정교화

## 고려 사항
- 조명 변화 대응
- 안경 착용 사용자 대응
- 장시간 실행 안정성
- 발열 및 성능 관리
- 오프라인 동작 가능 여부

- 책상 위 배치 형태와 외형 방향
